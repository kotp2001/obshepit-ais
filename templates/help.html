Руководство пользователя АИС "Общепит"

По ГОСТ 19.505-79 ЕСПД

1. Назначение системы

АИС "Общепит" предназначена для автоматизации учёта заказов на предприятиях общественного питания (кафе, рестораны, столовые). Доступ только для сотрудников: администраторов, официантов и поваров.

Функции администратора:
  • Управление базой блюд и составом меню
  • Управление столами (статус, количество мест)
  • Управление сотрудниками
  • Просмотр и экспорт отчётов в Excel
Функции официанта:
  • Просмотр схемы зала с цветовой индикацией
  • Создание заказа с выбором блюд по категориям
  • Получение уведомлений о готовых заказах
  • Приём оплаты (наличные, карта, QR-код)
  • Формирование и печать чека
Функции повара:
  • Просмотр активных заказов
  • Изменение статуса блюд: "Ожидает" → "Готовится" → "Готов"
  • Отправка уведомления о готовности заказа

2. Условия выполнения программы

Минимальные требования к серверной части:
  • Процессор: 2 ядра, 2 ГГц
  • Оперативная память: не менее 4 Гб
  • Свободное дисковое пространство: не менее 10 Гб
  • Стабильное подключение к сети Интернет
Минимальные требования к клиентской части:
  • Компьютер с ОС Windows / Linux / macOS
  • Современный браузер (Chrome, Firefox, Edge)
  • Доступ к сети Интернет
  • Разрешение экрана не менее 1366×768

  2. Вы увидите схему зала с цветовой индикацией столов

4.2. Приём заказа
  1. Нажмите на свободный зелёный стол в схеме зала
  2. В открывшемся окне выберите блюда из меню, нажимая кнопку "+"
  3. Укажите количество гостей
  4. Нажмите "Оформить заказ"

  2. Нажмите "Забрать заказ"
  3. Выберите способ оплаты: Наличные / Карта / QR-код

5. Работа повара

  1. Войдите в систему как повар
  2. В разделе "Активные заказы" вы увидите список заказов
  3. У каждого блюда нажмите "Начать готовку" (статус "Готовится")
  4. После приготовления нажмите "Готово" (статус "Готов")
  5. Когда все блюда готовы, нажмите "Заказ готов"

6. Работа администратора

  1. Войдите в систему как администратор
  2. В панели администратора доступны карточки:
    • Меню - управление категориями и блюдами
    • Столы - управление столами
    • Сотрудники - управление пользователями
    • Заказы - просмотр всех заказов
    • Бронирования - управление бронями
    • Отчёты - статистика и экспорт
